
# Tutorial correr proyecto 


Este proyecto incluye dos comandos principales para iniciar la aplicación backend:

- `npm run start:dev`: Inicia el servidor en modo desarrollo utilizando `nodemon`, lo que permite la recarga automática cuando se detectan cambios en los archivos fuente.
- `npm run start:prod`: Inicia el servidor en modo producción ejecutando directamente `index.js` con Node.js.

Asegúrate de tener instaladas las dependencias ejecutando `npm install` antes de iniciar el proyecto.
